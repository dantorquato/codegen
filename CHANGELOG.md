## [1.0.0] - 2025-11-07

### Added
- Initial release
- Template-based code generation with metadata support
- Cross-platform standalone executables (Windows, Linux, macOS)
- Variable substitution (PascalCase, camelCase, UPPERCASE, kebab-case)
- File protection (never overwrites existing files)
- Three production-ready C# templates (Model, Service, Controller)
- 15 automated tests with xUnit
- Build scripts for all platforms
- Comprehensive documentation
- MIT License

### Features
- ✅ Template processor with metadata extraction
- ✅ Variable replacer with multiple case styles
- ✅ Code generator with safe file operations
- ✅ CLI interface
- ✅ Self-contained executables (~12MB)
- ✅ Support for any programming language templates

### Documentation
- ✅ README with quick start guide
- ✅ CONTRIBUTING guidelines

- ✅ Build and usage examples
